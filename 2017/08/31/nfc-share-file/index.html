发送文件到其他设备

Android Bean 文件传输功能有以下要求:

  1. Android4.1以上支持大文件传输
  2. 只有放在外部存储中的文件才可以传输
  3. 传输的文件必须是 world-readable.可以通过File.setReadable)设置权限
  4. 必须提供文件的URI,Android Beam 文件传输无法处理通过FileProvider.getUriForFile) 生成的内容URIs
